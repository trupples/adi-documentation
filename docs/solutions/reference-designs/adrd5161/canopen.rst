ADRD5161 CANopen reference
==========================

The current version of the ADRD5161 module exposes the following parameters via the CAN interface:

* OD_RAM.x2060_batteryCellVoltages[0-3]
* OD_RAM.x2071_current
* OD_RAM.x6000_batteryStatus
* OD_RAM.x6001_chargerStatus
* OD_RAM.x6010_temperature
* OD_RAM.x6052_ahReturnedDuringLastCharge
* OD_RAM.x6060_batteryVoltage
* OD_RAM.x6080_chargerStateOfCharge
* OD_RAM.x6081_batteryStateOfCharge
* OD_RAM.x6070_chargeCurrentRequested
